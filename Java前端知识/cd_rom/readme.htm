配套光盘使用指南

一、本光盘包括内容
本光盘包括本书所有的源代码。 所有程序均在Windows XP/2003+JDK1.5 +Tomcat5.5调试通过。
二、使用方法

1.按照本书第1章和第二章中介绍的方法安装与配置。

2.本书的源代码一般分章依次存放，Chapter 01目录下存放第一章所有源程序，以此类推。

3.因为从光盘复制的文件属性是只读的，所以数据库文件和文本文件是需要读写的。在使用前，将所有文件和文件夹的只读属性去掉,否则将造成某些程序不能执行。

4. 在Ecllpse中打开光盘中工程的方法：
将光盘的内容拷贝到硬盘，并去掉所有文件的只读属性，选择[File] -> [Import]菜单

选择"Existing Projects into Workspace".

找到刚才从光盘上拷贝过来的目录,选某个工程加入到集成开发环境中.

如果工程需要发布,请参考第二章的发布步骤.
三、注意事项

1. 访问光盘中文命名的文件夹，有的系统对中文文件夹支持的不好， 如果不能访问， 请将中文名称去掉。

2. 如果执行过程中出现文件不能写，或者数据库文件不能写的情况，请将文件的只读属性去掉。

3.本光盘内不是所有程序都可以单独运行，有些文件只能被其他文件调用，但是按照“章节-编号.jsp或者Java”命名的文件都可以独立执行。
