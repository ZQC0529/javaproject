關於 urDrive

 

urDrive 是為 Kingston DataTraveler® 隨身碟設計的專屬預載軟體，能讓您更有效率地在隨身碟中搜尋及管理所需資料。 除了藉由時尚的介面來管理您的資料，urDrive 也採用全新的方法來保護您的電腦並提升效能。

*注意： urDrive 僅適用於指定的 DataTraveler® 隨身碟產品。 欲了解更多 urDrive 最新資訊，請至 www.kingston.com/urdrive

 

 

系統需求

 

 

電腦平台

 

·         Intel Atom 處理器 (或更高階)

·         150MB 的隨身碟可用空間

·         支援 Java

·         urAppZone 及 urDrive 的精選解決方案需要使用網際網路服務

 

 

 

 

 

 

 

 

 

作業系統支援

 

·            Windows® 7

·            Windows® Vista SP1，SP2

·            Windows® XP SP3

 

(注意： 為達到最佳效能，系統至少需配備 2GB 的 RAM。)

 

開始使用

1.    點選USB 隨身碟上的 -urDrive.exe 檔案來啟動 urDrive 軟體。

2.    從下拉式選單中選取您偏好的語言。 圖 1.1。

3.    按一下「開始使用」按鈕，繼續前往「使用者授權合約」。

4.    您可選擇： 「接受條款與條件」，「拒絕並關閉」，或是「拒絕並解除安裝」。 為能使用本軟體，請選擇「接受條款與條件」，然後按一下「下一步」繼續操作。

主畫面 

 

「主畫面」可讓您使用 urDrive 的每一項功能。 您可以在此建立 urDrive 帳戶，了解「解決方案」，並藉由「導覽列」來加入更多內容，體驗 urDrive 提供的每一項功能。

  

解決方案

 

 

「PC Checkup」功能可讓您下載免費的工具軟體，自動掃描您的電腦及檢查是否有潛在的安全性風險或效能問題。 同時有助於改善安全性，並讓您的電腦維持優異的效能。 欲了解更多資訊，請至  http://us.norton.com/nortonlive/free-pc-checkup.jsp

 

「瀏覽」功能可讓您在使用 Maxthon 的 Dual Display Engines 「遨遊」功能，以超快速度的體驗網際網路，並可在 Ultra 模式與 Retro 模式間進行切換動作。  欲了解更多資訊，請至 www.maxthon.com/help-and-support

 

「備份」功能可讓您根據 urDrive 的備份公用程式 vdrive 中所指定的使用者喜好設定，來備份與復原資料。  您也可透過選用的線上備份功能來備份您的資料，讓您在任何瀏覽器上皆能安全地存取資料。 欲了解更多資訊，請至 http://skyfile.maxthon.com/help.html

 

「For Kids」可讓家長管控子女們在網路上存取的內容，給予孩童更安全的上網環境。 藉由自訂面板功能，您可根據年齡，性別或興趣來限制可存取的內容。 欲了解更多資訊，請至  www.urfooz.com/fooz-kids/account/support/request

 

「遊戲」提供了有趣新潮的遊戲供您試玩。您可以透過「商店」連結下載其他的遊戲。

 

「商店」連結可讓您進入 urAppZone，存取許多可供下載的遊戲，應用程式及公用程式。

 

Facebook 連結可讓您透過內嵌的 Maxthon 瀏覽器，快速存取 Facebook 登入畫面。

 

YouTube 連結可讓您連線至 Youtube 首頁。您可以在此處登入帳戶或搜尋眾多由使用者自行創作的視訊。

 

Twitter 連結是 Twitter 首頁的捷徑。您可以在此處建立帳戶或登入現有帳戶。

 

導覽列

 

 

 

使用者只需按一下「首頁」圖示，即可從 urDrive 應用程式中的任何位置返回到首頁。

 

「我的檔案」圖示讓您存取儲存在 urDrive (及子資料夾) 中的所有檔案。  不同於「我的相片」，「我的音樂」及「我的視訊」，「我的檔案」不會作特定的檔案類型分類，因此所有在urDrive中的檔案內容皆會顯示出來。 欲了解更多資訊，請參閱本文件第 12 頁「了解『我的檔案』」的部分。

 

「我的相片」圖示讓您快速簡單地存取儲存在 urDrive (Photos 資料夾中) 的相片。 僅有「我的相片」支援的檔案類型才會顯示於此。  [支援： .jpeg，.jpg，.bmp，.gif，.cdr，.psd] 欲了解更多資訊，請參閱本文件第 13 頁「了解『我的相片』」的部份。

 

「我的音樂」圖示讓您輕鬆存取在 urDrive (Music 資料夾中) 的音樂。  僅有「我的音樂」支援的檔案類型才會顯示於此。 支援： .mp3] 欲了解更多資訊，請參閱本文件第 15 頁「了解『我的音樂』」的部分。

 

「我的視訊」圖示讓您方便快速地存取儲存於 urDrive (Videos 資料夾中) 的視訊檔案。  僅有「我的視訊」支援的檔案類型才會顯示於此。  支援： .avi，.asf，.mpeg，.mpg，.wmv，.rmvb，.rm，.dat，.mp4，.mov] 欲了解更多產品資訊，請參閱本文件第 18 頁「了解『我的視訊』」的部分。

 

「新聞」圖示讓使用者存取所有 Kingston 產品的最新資訊。

 

「設定」圖示讓您自訂使用者內容，包括語言，背景圖案及個人檔案等。 欲了解更多資訊，請參閱本文件第 20 頁「了解我的設定」的部分。

 

「說明」圖示能讓您檢視及列印本產品的《使用者指南》，並透過瀏覽 Kingston 網站來取得更多的支援，請至：http://www.kingston.com/support/urDrive

將內容加入 UrDrive

 

urDrive 支援「拖曳式」功能，幫助您輕鬆整理所有檔案。

 

利用 Windows 的檔案總管，將特定的檔案複製到所屬的資料夾以增加 urDrive 的內容，例如將 .jpg 加入 Photos 資料夾，將 .mp3 檔案加入 Music 資料夾等等。 請參閱圖 2.1。

 

藉由這種整理方式，您將能夠輕鬆管理所有檔案，並使其在 urDrive 中井然有序地呈現。

 

 

圖 2.1   拖曳式

 

請將常使用的檔案放在 urDrive 桌面上的「我的最愛」資料夾。

 

按一下「我的最愛」按鈕，並將檔案從 Windows 檔案總管介面中拖入「我的最愛」視窗。 請參閱圖 2.2。

 

 

圖 2.2 - 我的最愛

 

 

 

只要將游標移到導覽連結上，即可顯示「我的檔案」，「我的相片」，「我的音樂」及「我的視訊」的內容，如圖 2.3 所示。*

 

當出現快顯功能表時，點兩下欲檢視的檔案或目錄 (我的檔案等等) 來顯示所有支援的內容。

 

(提醒： 「我的檔案」會顯示所有的檔案內容 (無論檔案類型為何)，而其他的連結僅會顯示與該類別相關的檔案類型。

 

 

 

圖 2.3  滑鼠停留顯示

 

您的「容量表」位於 urDrive 主畫面桌面的左下角。 您的隨身碟可使用的儲存容量,會同時以數字與圖像的方式顯示。 請參閱圖 2.4。

 

 

 

 

圖 2.4  容量表

 

當您將檔案加進裝置後，此量表的指標會增加，直到空間已滿。

 

若在 urDrive 中放入過多資料，您可能會收到圖 2.5 所示的警訊。

 

要解決此問題，請關閉 urDrive，刪除一或多個未使用的檔案，再重新開啟 urDrive。

 

 

圖 2.5.  在 urDrive 中放入過多資料

 

 

*注意： 滑鼠停留功能會顯示主畫面上其導覽連結內所儲存的內容。 如果您目前正在瀏覽某個連結 (我的相片，我的音樂等等) 內的內容，按一下滑鼠即會顯示與該類別相關的檔案類型。

 了解「我的檔案」

 

 

「我的檔案」圖示可讓您存取儲存在 urDrive (及其子資料夾) 中所有的檔案內容。urDrive 會利用您預設的 Microsoft Windows 程式來開啟在「我的檔案」視窗中所選定的檔案。

圖 5.2  幻燈片秀工具列

在預設情況下，磁碟的根目錄存有以下檔案：

 

·         autorun.inf

·         unInstaller.exe

·         urDrive.exe
