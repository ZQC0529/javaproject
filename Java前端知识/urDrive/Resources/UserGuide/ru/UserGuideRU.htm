Программное обеспечение urDrive

 

urDrive — это предустановленное программное обеспечение, созданное специально для USB-накопителей Kingston DataTraveler®*, которое предоставляет новые возможности для просмотра и управления данными, записанными на накопителе. ПО urDrive имеет новый привлекательный интерфейс, позволяет систематизировать данные с помощью рабочего стола и предлагает новые способы для защиты и оптимизации работы компьютера.

Примечание: программное обеспечение urDrive доступно только для отдельных моделей накопителей DataTraveler®. Для получения дополнительной информации о ПО urDrive посетите веб-сайт www.kingston.com/urdrive.

 

 

Системные требования

 

 

 

Аппаратная платформа

 

 

·         Процессор Intel Atom (или более быстрый)

·         150 Мб свободного пространства на USB-накопителе

·         Веб-браузер с поддержкой Java

·         Для использования решений на основе urAppZone и отдельных функций urDrive требуется доступ в сеть Интернет

 

 

 

 

 

 

 

 

 

 

Поддерживаемые операционные системы

 

·            Windows® 7

·            Windows® Vista (SP1, SP2)

·            Windows® XP (SP3)

 

(Примечание: для обеспечения оптимальной работы рекомендуется использовать 2 Гб оперативной памяти.)

 

 

Начало работы

1.    Запустите программное обеспечение urDrive, дважды щелкнув по файлу urDrive.exe, расположенному на USB-накопителе.

2.    Выберите язык интерфейса из раскрывающегося меню. См. Рисунок 1.1.

3.    Нажмите кнопку Get Started (Начало работы), чтобы перейти к лицензионному соглашению об использовании ПО .

4.    Доступны следующие варианты действий: Accept the terms and conditions (Принять условия и положения), Decline and close (Отклонить условия и положения и закрыть) или Decline and uninstall (Отклонить условия и положения и удалить ПО). Чтобы получить право на использование программного обеспечения, выберите Accept Terms & Conditions (Принять условия и положения) и нажмите кнопку Next (Далее), чтобы продолжить.

Программные решения 

 

 

 

Функция PC Checkup позволяет загрузить бесплатное программное приложение, которое автоматически проведет проверку вашего компьютера на предмет наличия потенциальных угроз безопасности и/или проблем с производительностью. Это решение также помогает повысить безопасность и производительность вашего компьютера. Для получения информации по вопросам технической поддержки посетите веб-сайт http://us.norton.com/nortonlive/free-pc-checkup.jsp.

 

Функция Browse (Обзор) позволяет выходить в сеть Интернет с использованием технологии Dual Display Engines™ браузера Maxthon, которая оптимизирует просмотр веб-сайтов за счет использования технологии сверхбыстрого отображения страниц и возможности переключения между режимами Ultra Mode (Ультра) и Retro Mode (Ретро). Для получения информации по вопросам технической поддержки посетите веб-сайт www.maxthon.com/help-and-support.

 

Функция Backup (Резервное копирование) позволяет создавать резервные копии и восстанавливать данные в соответствии с установленными вами параметрам в утилите резервного копирования vdrive . Пользователь также может создать резервную копию данных с помощью дополнительной функции онлайнового резервного копирования, которая обеспечивает безопасный и надежный доступ к данным из любого веб-браузера. Для получения информации по вопросам технической поддержки посетите веб-сайт http://skyfile.maxthon.com/help.html.

 

Fooz Kids — это приложение, которое создает безопасную и защищенную среду для детей при работе в сети Интернет, предоставляя, при этом, родителям возможность контролировать и отслеживать, какие веб-сайты просматривают их дети. С помощью настраиваемых панелей можно разрешить или запретить конкретные типы веб-сайтов в зависимости от возраста, пола или увлечений ребенка. Для получения информации по вопросам технической поддержки посетите веб-сайт www.urfooz.com/fooz-kids/account/support/request.

 

 

По ссылке Games (Игры) можно получить доступ к списку новых увлекательных игр. Также можно загрузить отдельные игры, перейдя по ссылке Store (Магазин).

 

 

Ссылка Store (Магазин) позволяет перейти на портал urAppZone, где можно найти сотни доступных для загрузки игр, приложений и других полезных утилит.

 

 

По ссылке Facebook™ можно быстро перейти на страницу входа в сеть Facebook через встроенный браузер Maxthon.

 

 

Сслыка YouTube™ позволяет перейти на домашнюю страницу портала Youtube. Отсюда можно зайти в свою учетную запись или осуществлять поиск многочисленных пользовательских видео.

 

 

Сслыка Twitter™ дает возможность быстрого доступа к домашней странице сети Twitter. Отсюда можно создать новую учетную запись или войти в уже существующую.

Значок My Photos (Мои фотографии) обеспечивает быстрый и простой доступ ко всем изображениям, которые хранятся на urDrive (в папке Photos (Фотографии)). В этом разделе отображаются только те типы файлов, которые поддерживаются папкой My Photos (Мои фотографии). [[Поддерживаемые типы файлов: .jpeg, .jpg, .bmp, .gif, .cdr, .psd] Для получения дополнительной информации см. раздел «Папка My Photos (Мои фотографии)» на стр. 13 этого документа.

 

 

Значок My Music (Моя музыка) обеспечивает быстрый и простой доступ ко всем аудиозаписям, которые хранятся на urDrive (в папке Music (Музыка)). В этом разделе отображаются только те типы файлов, которые поддерживаются папкой My Music (Моя музыка). [Поддерживаемые типы файлов: .mp3] Для получения дополнительной информации см. раздел «Папка My Music (Моя музыка)» на стр. 15 этого документа.

 

 

Значок My Videos (Моё видео) обеспечивает быстрый и простой доступ ко всем файлам видеозаписей, которые хранятся на urDrive (в папке Video (Видеозаписи)). В этом разделе отображаются только те типы файлов, которые поддерживаются папкой My Videos (Моё видео). Добавление контента на UrDrive

 

urDrive поддерживает функцию перетаскивания для простого управления файлами.

 

Перенесите все необходимые файлы в urDrive с помощью удобного интерфейса Windows Explorer, скопировав файлы определенных типов в соответствующие папки, например, файлы JPG в папку Photos (Фотографии), файлы .mp3 в папку Music (Музыка) и так далее. См. Рисунок 2.1.

 

Этот способ систематизации файлов обеспечит простой доступ к ним и их организованное размещение.

 

 

 

Рисунок 2.1 — Операция перетаскивания

 

 

Для наиболее популярных файлов используйте папку Favorites (Избранное), расположенную на рабочем столе urDrive.

 

Нажмите кнопку Favorites (Избранное) и перетащите файлы из Проводника Windows в окно Favorites (Избранное). См. Рисунок 2.2.

 

 

 

Рисунок 2.2 — Папка Favorites (Избранное)

 

 

 

Можно просмотреть содержимое папок My Files (Мои файлы), My Photos, (Мои фотографии), My Music (Моя музыка) и My Videos (Моё видео), наведя курсор мыши на ссылку для навигации, как показано на Рисунке 2.3.*

                                

После отображения всплывающего окна можно дважды щелкнуть файл, который необходимо просмотреть, или дважды щелкнуть каталог (My Files (Мои файлы) и т. п.), чтобы отобразить весь поддерживаемый контент, размещенный в этих папках.

 

Примечание: в разделе My FIles (Мои файлы) отображаются все файлы на устройстве, независимо от их типа, в то время как остальные навигационные ссылки будут отображать только поддерживаемые типы файлов, связанные с этой категорией.

 

 

 

Рисунок 2.3 — Меню, отображаемое при наведении курсора мыши

 

Раздел Capacity Gauge (Индикатор емкости) находится в нижнем левом углу рабочего стола главной страницы urDrive. Информация о занятом или свободном пространстве на USB-накопителе отображается как в численном, так и в графическом виде. См. Рисунок 2.4.

 

 

 

 

Рисунок 2.4 — Индикатор емкости

 

 

При добавлении новых файлов этот индикатор будет показывать все большее значение до тех пор, пока накопитель не будет полностью заполнен.

 

В случае загрузки файлов, которые не могут быть сохранены по причине отсутствия необходимого свободного пространства, на экране будет показано сообщение, представленное на Рисунке 2.5.

 

В этом случае необходимо закрыть urDrive, удалить один или несколько неиспользуемых файлов и запустить urDrive снова.

 

 

Рисунок 2.5 — Переполнение urDrive

 

Примечание: функция отображения меню при наведении курсора мыши позволяет показать контент, который хранится в каждой из соответствующих навигационных ссылок, но только для ссылок, доступных в главном окне программы. Если на экране отображается контент, полученный после перехода по ссылке (из папок My Photos (Мои фотографии), My Music (Моя музыка) и т. п.), то одним нажатием кнопки мыши можно показать типы файлов, связанные с этой категорией.

Регистрация учетной записи urDrive

(Примечание: если у вас уже есть действующая учетная запись urDrive, вы можете пропустить этот шаг и перейти к разделу Sign In (Регистрация).)

Щелкните ссылку Sign Up (Регистрация), расположенную в правом верхнем углу главного окна.

В ходе регистрации пользователь должен указать адрес электронной почты, имя, фамилию и предпочитаемый пароль. Введите необходимые сведения в соответствующие поля, как показано на Рисунке 3.1.

Примите Terms of Service (Условия предоставления услуг) и нажмите Sign Up (Зарегистрироваться) для завершения процедуры.

Пользователь может дать согласие на получение новостей о ПО urDrive по электронной почте, установив соответствующий флажок. (Это поле не является обязательным.)
