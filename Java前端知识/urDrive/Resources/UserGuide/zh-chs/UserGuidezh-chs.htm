关于 urDrive

 

urDrive 是金士顿 DataTraveler® USB 闪存盘的专用预装软件*，可让您以一种令人愉快的全新方式浏览并管理 USB 闪存盘中的内容。除了其时尚的新界面和简洁易用的程序桌面以外，urDrive 还引入了新的方法来帮助保护和提高您的计算机性能。

*注意：仅特定 DataTraveler 产品提供 urDrive 功能。有关 urDrive 的最新信息，请访问我们的 WEB 地址：www.kingston.com/urdrive。

 

 

系统要求

 

 

PC 平台

 

·         英特尔 Atom（或更快速度的）处理器

·         USB 闪存盘中有 150MB 可用空间

·         启用 Java 功能的浏览器

·         urAppZone 和 urDrive 的特色解决方案需要互联网连接

 

 

 

 

 

 

 

 

 

PC 操作系统支持

 

·            Windows® 7

·            Windows® Vista SP1，SP2

·            Windows® XP SP3

 

（注意：为了实现最佳性能，建议使用 2GB 的系统内存。）

 

 

开始

1.    双击位于 USB 闪存盘中的 urDrive.exe 文件，启动 urDrive 软件。

2.    从下拉菜单中选择您的首选语言。图 1.1。

3.    单击 启动按钮继续浏览直至最后用户许可协议。

4.    您可以选择：“接受条款”，“拒绝并关闭”或“拒绝并卸载”。要使用该软件，请选择接受条款并单击下一步继续。

 

 

 

图 1.1 – Welcome 窗口

导航栏

 

 

 

 

用户只需通过单击主页图标即可从 urDrive 应用程序内的任何位置返回到主屏幕。

 

通过我的文件图标，可以访问存储在 urDrive（以及子文件夹）内的全部内容，没有文件类型或程序扩展名的限制。不同于“我的照片”，“我的音乐”和“我的视频”，我的文件并不与特定的文件类型相关联，因此会显示 urDrive 的全部内容，而不会进行过滤。有关其他信息，请参阅本文档第 12 页的“浏览‘我的文件’”。

 

 

通过我的照片图标，可以方便快速地访问存储在 urDrive（“照片”文件夹）内的全部图片内容。只有我的照片支持的文件类型才会在此部分中显示。[支持的文件类型： .jpeg, .jpg, .bmp, .gif, .cdr, .psd] 有关其他信息，请参阅本文档第 13 页上的“浏览‘我的照片’”。

 

 

通过我的音乐图标，可以方便快速地访问存储在 urDrive（“音乐”文件夹）内的全部音乐内容。只有“我的音乐”支持的文件类型才会在此部分中显示。[支持的文件类型：.mp3] 有关其他信息，请参阅本文档第 15 页的“浏览‘我的音乐’”。

 

 

通过我的视频图标，可以方便快速地访问存储在 urDrive（“视频”文件夹）内的全部视频内容。只有“我的视频”支持的文件类型才会在此部分中显示。[支持的文件类型：.avi, .asf, .mpeg, .mpg, .wmv, .rmvb, .rm, .dat, .mp4, .mov] 有关其他信息，请参阅本文档第 18 页的“浏览 ‘我的视频’”。

 

 

新闻图标使用户可以访问关于金士顿产品的全部最新信息。

 

设置图标通过提供对 urDrive 上可用的一些界面首选设置（包括语言选择，背景和显示图片）的访问，使您可以定制用户体验。有关其他信息，请参阅本文档第 20 页的“浏览‘我的设置’”。

 

帮助图标使您可以查看和/或打印该产品的用户指南，同时也可访问金士顿网站来获取其他支持信息，WEB 地址为：http://www.kingston.com/support/urDrive。

 

导航栏快速指引

 

 

向 UrDrive 增加内容

 

urDrive 支持“拖放功能”，可实现简单文件管理。 

 

使用 Windows 资源管理器界面，通过复制特定文件类型到其相应的文件夹，即 .jpg 文件到 照片文件夹，.mp3 文件到音乐文件夹等来填充 urDrive。请参阅图 2.1。

 

通过这种方式管理文件，您可以轻松访问您的文件，并使 urDrive 中的文件保持良好的结构和顺序。

 

 

图 2.1  – 拖放

 

对于经常访问的文件，请使用 urDrive 桌面上的收藏文件夹。

 

单击收藏夹按钮，并将文件从 Windows 资源管理器界面拖到收藏夹窗口中。请参阅图 2.2。

 

 

 

图 2.2 - 收藏夹

 

 

 

只需将鼠标指针放在导航链接上即可显示我的文件，我的照片，我的音乐以及我的视频中的内容，如图 2.3所示。*

 

弹出窗口出现之后，可以双击希望查看的文件或双击目录（“我的文件”等）显示这些位置中所有支持的内容。

 

（提示：“我的文件”将显示全部文件内容，不区分文件类型，而其余导航链接将仅显示与该类别相关联的，所支持的文件类型。

 

 

 

图 2.3 – 鼠标悬停显示

 

您的存储容量指示位于 urDrive 主界面的左下角。以数字和图形形式显示您的 USB 闪存盘的剩余（或可用）存储容量。请参阅图 2.4。

 

 

 

 

 

图 2.4 – 存储容量指示

 

 

随着您不断地向设备中添加内容，指示数字也会不断变大，直到闪存盘达到其最大存储容量。

 

如果您尝试向 urDrive 中添加过量的内容，会收到警告信息，见图 2.5。

 

要解决这一问题，请关闭 urDrive，删除一个或多个不使用的文件，并重新启动 urDrive。

 

 

图 2.5 – 向 urDrive 中添加了过多内容

 

 

*注意：鼠标悬停功能仅在主屏上显示存储在各对应导航链接内的内容。如果正在显示一个链接中的内容（“我的照片”，“我的音乐”等），则单击一次鼠标将显示与该类别相关联的文件类型。
