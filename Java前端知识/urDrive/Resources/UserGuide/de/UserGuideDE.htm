Über urDrive

 

Der USB-Flashspeicher* DataTraveler® von Kingston Technology verfügt exklusiv über die bereits installierte Software urDrive, die Ihnen eine tolle neue Möglichkeit bietet, den Inhalt Ihres USB-Flashspeichers anzuzeigen und zu verwalten. Über seine elegante neue Benutzeroberfläche und den Desktop-Ansatz zur Organisation Ihrer Daten hinaus bietet urDrive außerdem neue Wege, die Leistung Ihres Computers zu schützen und zu verbessern.

*Note: urDrive ist nur auf bestimmten DataTraveler® Produkten verfügbar. Die aktuellsten Informationen zu urDrive finden Sie im Internet unter www.kingston.com/urdrive.

 

 

Systemanforderungen

 

 

PC-Plattform

 

·         Intel Atom Prozessor (oder schneller)

·         150 MB freier Speicherplatz auf dem USB-Flashspeicher

·         Java-fähiger Browser

·         Für die Top-Anwendungen von urAppZone und urDrive ist ein Internetzugang erforderlich.

 

 

 

 

 

 

 

 

 

Unterstützte PC-Betriebssysteme

 

·            Windows® 7

·            Windows® Vista SP1, SP2

·            Windows® XP  SP3

 

(Hinweis: Für eine optimale Leistung werden 2 GB RAM empfohlen).

 

Erste Schritte

1.    Starten Sie die urDrive Software durch Doppelklickauf das Dateisymbol urDrive.exe auf Ihrem USB-Speicher.

2.    Wählen Sie aus dem Dropdown-Menü Ihre bevorzugte Sprache. Abbildung 1.1.

3.    Klicken Sie auf die Taste Erste Schritte, um mit der ENDBENUTZER-LIZENZVEREINBARUNG  fortzufahren.

4.    Sie haben folgende Auswahlmöglichkeiten: ‚Nutzungsbedingungen akzeptieren‘, ‚Ablehnen und schließen‘ oder ‚Ablehnen und deinstallierenl‘. Um die Software zu verwenden, wählen Sie Nutzungsbedingungen akzeptieren und klicken auf WEITER, um fortzufahren.

 Startbildschirm 

 

 

Ihr Startbildschirm bietet Zugriff auf alle Aspekte des urDrive. Von hier aus können Sie ein urDrive-Benutzerkonto anlegen, die „Top-Anwendungen“ durchsuchen, über den „Navigationsbalken“ Inhalte hinzufügen und alles erfahren, was urDrive zu bieten hat.

 

 

 

Top-Anwendungen

  

 

 

Die Funktion PC Checkup ermöglicht Ihnen, eine kostenlose Software herunterzuladen, die Ihren Computer automatisch auf potenzielle Sicherheitsrisiken und/oder Leistungsprobleme hin scannt. Außerdem verbessert Sie die Sicherheit und sorgt dafür, dass Ihr Computer funktioniert, als sei er neu. Support-Informationen finden Sie unter http://us.norton.com/nortonlive/free-pc-checkup.jsp.

 

Die Funktion Durchsuchen ermöglicht Ihnen, mit den  Dual Display Engines™, von Maxthon im Internet zu surfen und Ihre Web-Erfahrung mit blitzschneller Technologie und der Möglichkeit, zwischen Ultra- und Retro-Modus umzuschalten, individuell anzupassen.  Support-Informationen finden Sie unter www.maxthon.com/help-and-support.

 

Die Funktion Sicherungskopie ermöglicht Ihnen, Daten entsprechend Ihrer Benutzereinstellungen in vdrive, dem speziellen Backup-Hilfsprogramm von urDrive, zu sichern und wiederherzustellen.  Sie können außerdem mit dem optionalen Online Backup eine zweite Kopie Ihrer Daten anfertigen, eine sichere Art, von jedem Webbrowser aus auf Ihre Daten zuzugreifen. Support-Informationen finden Sie unter http://skyfile.maxthon.com/help.html.

 

Fooz Kids ist eine Anwendung, die Kindern eine sichere Internetumgebung bietet und Eltern Hilfsmittel und die Möglichkeit bietet, zu steuern und zu überwachen, worauf ihre Kinder online zugreifen. Mittels eines anpassbaren Dashboards können Sie bestimmte Inhalte abhängig von Alter, Geschlecht oder Interessen zulassen oder verbieten. Support-Informationen finden Sie unter www.urfooz.com/fooz-kids/account/support/request.

 

Der Link Spiele bietet eine Menge Spaß und Unterhaltung mit einer Liste toller neuer Spiele zum Ausprobieren. Mit dem Link Store können Sie weitere Spiele herunterladen.

 

Der Link Store dient als Portal für die urAppZone, über das Sie auf Hunderte von Spielen, Anwendungen und Hilfsprogramen Zugriff erhalten.

 

Der Link Facebook™ bietet über den integrierten Maxthon Browser schnellen Zugriff auf den Facebook Loginbildschirm

 

 

Der Link YouTube™ verbindet Sie zur YouTube Startseite. Von hier aus können Sie in Ihr Benutzerkonto einloggen oder nach zahlreichen von Benutzern erstellten Videos suchen.

 

Der Link Twitter™ dient dem Direkzugriff auf die Twitter Startseite. Von hier aus können Sie ein Benutzerkonto erstellen oder in ein vorhandenes Benutzerkonto einloggen.

 

Navigationsbalken

 

 

 

Das Symbol Startseite bietet Nutzern die Möglichkeit, mit einem einzigen Klick von jedem Ort in der urDrive Anwendung zum Startbildschirm zurückzukehren.

 

Das Symbol Meine Dateien ermöglicht den Nutzern den Zugriff auf alle Inhalte, die auf urDrive (und in dessen Unterordnern) gespeichert sind, unabhängig von Dateityp oder Programmerweiterung. Im Gegensatz zu Meine Fotos, Meine Musik und Meine Videos ist Meine Dateien nicht an einen bestimmten Dateityp gebunden und zeigt darum ungefiltert alle Inhalte des urDrive an. Weitere Informationen finden Sie unter Durchsuchen von „Meine Dateien“ auf Seite 12 dieses Dokuments.

 

Das Symbol Meine Fotos bietet eine schnelle und einfache Art, auf alle Bilder zuzugreifen, die auf urDrive (im Ordner Fotos) gespeichert sind.  In diesem Abschnitt werden nur die Dateitypen dargestellt, die von Meine Fotos unterstützt werden.  [Unterstützte Dateitypen: .jpeg, .jpg, .bmp, .gif, .cdr, .psd] Weitere Informationen finden Sie unter Durchsuchen von „Meine Fotos“ auf Seite 13 dieses Dokuments.

 

Das Symbol Meine Musik bietet eine schnelle und einfache Art, auf alle Musikinhalte zuzugreifen, die auf urDrive (im Ordner Musik) gespeichert sind.  In diesem Abschnitt werden nur Dateitypen angezeigt, die von Meine Musik unterstützt werden.  [Unterstützte Dateitypen: .mp3] Weitere Informationen finden Sie unter Durchsuchen von „Meine Musik“ auf Seite 15 dieses Dokuments.

 

Das Symbol Meine Videos bietet eine schnelle und einfache Art, auf alle Videos zuzugreifen, die auf urDrive (im Ordner Videos) gespeichert sind. In diesem Abschnitt werden nur Dateitypen unterstützt, die von Meine Videos unterstützt werden. [Unterstützte Dateitypen: .avi, .asf, .mpeg, .mpg, .wmv, .rmvb, .rm, .dat, .mp4, .mov] Weitere Informationen finden Sie unter Durchsuchen von „Meine Videos“ auf Seite 18 dieses Dokuments.

 

Das Symbol Nachrichten bietet Nutzern Zugriff auf die neuesten Informationen zu Produkten von Kingston.

 

Das Symbol Einstellungen ermöglicht Ihnen die Anpassung Ihrer Benutzererfahrung durch Zugriff auf einige der auf urDrive verfügbaren Benutzereinstellungen, darunter Sprachauswahl, Hintergrund und Profilbild. Weitere Informationen finden Sie unter Durchsuchen von „My Settings“ auf Seite 20 dieses Dokuments.

 

Das Symbol Hilfe ermöglicht Ihnen, den Anwenderleitfaden für dieses Produkt anzuzeigen bzw. auszudrucken und zusätzliche Unterstützungsinformationen zu erhalten, indem Sie Kingston im Internet unter folgender Adresse besuchen: http://www.kingston.com/support/urDrive.

 

Tipps zur Benutzung des Navigationsbalkens

 

 

Hinzufügen von Inhalten auf dem UrDrive

 

urDrive unterstützt „Drag-und-Drop“-Funktionalität für einfache Verwaltung von Dateien.

 

Sie können urDrive auch über die Benutzeroberfläche von Windows Explorer füllen, indem Sie bestimmte Dateien in die jeweiligen Ordner kopieren, d. h. Dateien in .jpg in den Ordner Fotos, Dateien in .mp3 in den Ordner Musik, und so weiter. Siehe Abbildung 2.1.

 

Wenn Sie Ihre Dateien auf diese Weise ordnen, sind sie auf Ihrem urDrive immer leicht zugänglich und übersichtlich.

 

 

Abbildung 2.1  – Drag und Drop

 

 

Benutzen Sie für häufig benutzte Dateien den Ordner Lesezeichen auf Ihrem urDrive Desktop.

 

Klicken Sie auf die Schaltfläche Lesezeichen und ziehen Sie Daten von der Oberfläche Ihres Windows Explorer in das Fenster Lesezeichen. Siehe Abbildung 2.2.

 

 

 

Abbildung 2.2 - Lesezeichen

 

 

 

Sie können den Inhalt von Meine Dateien, Meine Fotos, Mein Musik und Meine Videos anzeigen, indem Sie einfach Ihren Mauszeiger über den Navigationslink platzieren, wie in Abbildung 2.3 dargestellt.*

 

Ein Popup-Menü wird angezeigt, und Sie können die Datei, die Sie anzeigen wollen, oder das Verzeichnis (Meine Dateien usw.) doppelklicken um alle unterstützten Inhalten an diesen Ablageorten anzuzeigen.

 

(Denken Sie daran: „Meine Dateien“ zeigt alle Dateien unabhängig vom Dateityp an, und die übrigen Navigationsverknüpfungen zeigen nur die unterstützten Dateitypen an, die mit der jeweiligen Kategorie verknüpft sind.

 

 

 

Abbildung 2.3 – Anzeige durch Überstreifen mit dem Mauszeiger

 

Ihre Kapazitätsanzeige befindet sich in der unteren linken Ecke Ihres urDrive-Startanzeige-Desktops. Die verbleibende (oder verfügbare) Kapazität wird sowohl numerisch als auch grafisch dargestellt. Siehe Abbildung 2.4.

 

 

 

 

Abbildung 2.4 – Kapazitätsanzeige

 

 

Diese Anzeige wird beim Hinzufügen von Inhalten zu Ihrem Gerät erhöht, bis Sie die volle Kapazität des Speichers erreicht haben.

 

Wenn Sie versuchen, den urDrive zu überfüllen, erhalten Sie möglicherweise die in Abbildung 2.5 dargestellte Warnmeldung.

 

Um dieses Problem zu beheben, schließen Sie den urDrive, löschen Sie eine oder mehrere nicht mehr benötigte Dateien und öffnen Sie den urDrive erneut.

 

 

Abbildung 2.5 – Überfüllter urDrive

 

 

 

*Hinweis: Die Funktion des Überstreifens mit der Maus zeigt nur vom Startbildschirm aus die Inhalte an, die in jedem der entsprechenden Navigationslinks gespeichert sind. Wenn Sie bereits Inhalte innerhalb eines Links anzeigen (Meine Fotos, Meine Musik, usw.), genügt ein Mausklick, um die mit dieser Kategorie verbundenen Dateitypen anzuzeigen.

Root Directory

 

Ihr neuer USB-Speicher enthält bestimmte Dateien und Ordner, die für den Betrieb des urDrive notwendig sind. Greifen Sie auf das Root Directory Ihres Speichers auf die gleiche Weise zu, wie Sie auf jeden anderen USB-Speicher zugreifen würden: Start>Computer (oder ‚Mein Computer‘) und navigieren Sie zu Kingston urDrive.

 

 

Ihr Speicher enthält standardmäßig im Root Directory die folgenden Dateien:

 

·         autorun.inf

·         unInstaller.exe

·         urDrive.exe

 

 

Siehe Abbildung 10.1.
